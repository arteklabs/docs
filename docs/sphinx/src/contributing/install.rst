Install
=======

Requirements
------------

* docker
* poetry

Getting Started
---------------

Clone the repository and install the dependencies with:

.. code-block:: bash

    docs $ poetry shell
    docs $ poetry install

Run
---

Install from source with:

.. code:: shell

   docs $ pip install -e .

Verify the install with:

.. code:: shell

   docs $ docs --version
   docs, version 1.0.0
