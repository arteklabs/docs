Publish
=======

About
-----

The section shows how to build and publish the python package to AWS Code Artifact.

Requirements
------------

* :doc:`build`

Getting Started
---------------

Build a distribution package do ``dist`` (notice the ``.whl`` distribution file):

.. code:: shell

   (dmacli-py3.8) poetry publish
